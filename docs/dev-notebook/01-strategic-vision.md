# 01 — Strategic Vision

> The "why" behind everything we build.

## Mission

Build the most advanced AI-native operations platform in the asphalt paving industry — keep the flagship for J. Worden & Sons forever, license stripped versions to competitors as a perpetual revenue stream.

## Core principle

> **Never sell the crown. Sell the picks-and-shovels to everyone climbing the same mountain.**

This is the same model used by:
- **Bloomberg** (keep terminal, sell market data)
- **Palantir** (keep Foundry, sell Apollo)
- **SpaceX** (keep flagship rockets, sell Starlink as service)
- **Ferrari** (keep race cars, sell road cars to fund the race team)
